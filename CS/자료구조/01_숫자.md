# 숫자

## 정수 int

- immutable => 변수와 객체 참조간에 차이 없음
- `int()` 메서드

```python
# 문자열 -> 정수
s = '11'
d = int(s)
print(d)
>>> 11

# int(문자열, 밑)
s = '11'
d = int(s, 2)  # 2진수
print(d)
>>> 3

# int(float)
d = int(3.5)
print(d)
>>> 3

# 문자열이 float형이거나 밑 범위의 수를 벗어나면 에러
int('3.5')
int('12', 2)
```



## 부동소수점

- immutable
- single precision => 32bit
